# RoveSoPnP_GUI

## Setup
1. Clone repo
2. Run ```git submodule update --init --recursive```
3. Install QT
4. Set environment variable named "QT" to QT's installation location. Example: ```~/Qt/6.9.3/gcc_64```
5. Install quill
6. Define environment variables QT_EMAIL and QT_PASSWD with the corresponding values for your Qt account.
